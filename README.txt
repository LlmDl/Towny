Towny Advanced는 LlmDl이 개발했습니다.

MC 1.8이 출시된 이후로 ElgarL에게서 프로젝트를 인계받았습니다. 과거에 개발을 하셨던 분들입니다: Shadeness, FuzzieWuzzie, ElgarL.
그리고 가끔씩 다른 개발자들의 도움도 받고 있습니다: dumptruckman, ole8pie, SwearWord, gravypod, andrewyunt 등.

Towny는 Shadeness가 지금은 사용되지 않는 hMod라는 서버 플랫폼용으로 처음 개발한 이후 지금까지 오랫동안 개발되고 있는 마인크래프트 플러그인 중 하나입니다.
WorldGuard보다 조금 늦게 나와버려서 유명한 플러그인 중에서 두 번째로 오래된 토지 보호 플러그인이죠.

릴리즈, 개발 빌드와 제가 유지보수/개발하는 모든 플러그인은 여기에서 찾아보실 수 있습니다...

* http://palmergames.com/towny

도움이 필요하시거나 채팅하기를 원하시면 Esper.net 네트워크의 #towny IRC 채널로 오세요. ( http://webchat.esper.net/?channels=towny ) [영어]
정식 릴리즈되지 않은 개발 빌드를 얻고 테스트를 도와주고 싶은 서버 관리자 분들은 Discord 서버로 와 주세요 ( https://discord.gg/gnpVs5m ) [영어]
